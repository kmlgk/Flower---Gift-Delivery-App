Bloomora Template Guide

Bloomora is a pure HTML5 + Tailwind CSS (CDN) + Vanilla JS template — no Node.js, npm, or build step. Unzip the package and double-click any .html file to preview it in your browser.

Quick start

  1. Unzip the template folder.
  2. Open index.html directly in Chrome, Firefox, Edge or Safari.
  3. Edit colors/fonts in assets/js/tailwind-config.js.
  4. Edit navigation in assets/js/components.js.
  5. Deploy the folder as-is to any static host (Netlify, Vercel, cPanel, S3...).

Shared Components

Every page includes two empty containers, populated at runtime by components.js:

<div id="site-header"></div>
...page content...
<div id="site-footer"></div>

Add a CTA banner or breadcrumb hero anywhere with:

<div data-breadcrumb data-title="Page Title" data-parent="Home" data-parent-href="index.html"></div>
<div data-cta data-title="..." data-sub="..." data-btn="..." data-href="..."></div>

Set <body data-page="home"> (or about / services / blog / pages / contact) so the header highlights the correct nav item.

Theming, Dark Mode & RTL

Dark mode and RTL toggles call Bloomora.toggleTheme() and Bloomora.toggleDir() (in main.js), persisted to localStorage. A small inline script in every page's <head> re-applies the saved preference before first paint to avoid flashing.

Brand colors, font families and custom animation keyframes all live in assets/js/tailwind-config.js — edit the primary/secondary/accent color scales to re-theme the entire template.

Animations

All animation libraries are loaded via CDN and initialized defensively in main.js (each block checks the library exists first), so any page only needs to include the <script> tags for the libraries it actually uses:

  • GSAP + ScrollTrigger — blob float loops, parallax (data-parallax), text reveal (data-split-text + SplitType)
  • Lenis — global smooth scroll
  • AOS — scroll reveal (data-aos)
  • Swiper — carousels (data-swiper='{"...": "..."}')
  • Typed.js — rotating hero text (data-typed='["a","b"]')
  • CountUp.js — animated stats (data-countup="250")
  • GLightbox — image lightbox (class glightbox)
  • Isotope — filterable grids (data-isotope-grid)

FAQ

Do I need Node.js or npm?

No — everything runs from static files. Tailwind compiles in the browser via its CDN build.

Can I use this with a backend?

Yes — forms use standard HTML inputs; wire the onsubmit handlers to your API or a form service of your choice.

Where do I change the logo?

Replace assets/images/logo-icon.svg and the "Bloomora" text inside the headerHTML() function in components.js.
